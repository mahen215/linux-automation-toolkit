# 🐳 Docker Automation Toolkit

Professional Bash scripts for automating Docker administration tasks.

## 📌 Features

- Docker Cleanup
- Image Builder
- Resource Monitoring
- Volume Backup
- Volume Restore
- Container Reports
- Health Checks
- Auto Restart
- Image Backup
- Log Viewer

---

## 📂 Included Scripts

| Script | Description |
|---------|-------------|
| docker_cleanup.sh | Remove unused Docker resources |
| docker_image_builder.sh | Build Docker images |
| docker_resource_monitor.sh | Monitor Docker resources |
| docker_volume_backup.sh | Backup Docker volumes |
| docker_volume_restore.sh | Restore Docker volumes |
| docker_container_report.sh | Generate container reports |
| auto_restart_container.sh | Restart stopped containers |
| container_health_check.sh | Check container health |
| docker_stats_monitor.sh | Display Docker statistics |
| image_backup.sh | Backup Docker images |
| logs_viewer.sh | View container logs |

---

## ▶️ Run

```bash
chmod +x *.sh

./docker_cleanup.sh
```
