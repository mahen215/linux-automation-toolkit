# 🐧 Linux Automation Toolkit

A collection of Bash scripts designed to automate common Linux system administration tasks.

## 📌 Features

- System Information
- Disk Usage Monitoring
- Memory Monitoring
- CPU Monitoring
- Service Status Check
- Website Monitoring
- Backup Automation
- Log Cleanup
- User Management

---

## 📂 Included Scripts

| Script | Description |
|---------|-------------|
| system-info.sh | Display complete system information |
| disk-monitor.sh | Monitor disk usage |
| memory-monitor.sh | Check memory usage |
| cpu-monitor.sh | Monitor CPU load |
| service-check.sh | Check service status |
| backup.sh | Backup files/directories |
| log-cleanup.sh | Remove old log files |
| website-monitor.sh | Check website availability |
| user-management.sh | Create and manage Linux users |

---

## ▶️ Run

```bash
chmod +x *.sh

./system-info.sh
```
